# 📊 Expenses Dashboard — FY 2025 (Excel)

An interactive financial dashboard built in Microsoft Excel to track and analyze
company expenses across departments, categories, and payment methods.


## 🔑 Key Features
- **KPI cards:** Total Expenses, Budget Allocated, Budget Remaining, Transactions,
  Approval Rate, Avg Expense
- **Expense by Department & Category** (bar + donut charts)
- **Budget vs Actual** comparison by department
- **Monthly Trend** analysis
- **Payment Method** breakdown
- **Interactive slicers** for Month and Department (fully filterable)

## 🛠️ Built With
- Microsoft Excel
- PivotTables & PivotCharts
- Slicers for interactivity
- Custom dashboard design & formatting

## 📈 What It Shows
This dashboard turns raw expense transactions into a clear, decision-ready view —
letting stakeholders filter by month/department and instantly see spend, budget
variance, and trends.
